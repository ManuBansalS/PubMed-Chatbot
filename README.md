# RAG PubMed

## Overview
A Retrieval-Augmented Generation (RAG) system for PubMed research papers.

## Features
- Document retrieval from PubMed
- Context-aware generation
- Semantic search capabilities

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```python
from rag_pubmed import RAGSystem

rag = RAGSystem()
results = rag.query("your research question")
```

## Requirements
- Python 3.8+
- See `requirements.txt` for dependencies
